The ps -l command (lowercase L) in Linux/Unix shows several columns in the output.   For example:

F S   UID   PID  PPID  C PRI  NI ADDR SZ WCHAN  TTY          TIME CMD
0 S  1000  3520  3516  0  80   0 -  3333 do_wai pts/0    00:00:00 bash
0 S  1000 16489  3520  0  80   0 -  2008 ia32_s pts/0    00:00:00 sleep

The second column begins with S, which indicates the process state.  This column may have the following values (along with their meanings):

D Uninterruptible sleep (usually IO)
R Running or runnable (on run queue)
S Interruptible sleep (waiting for an event to complete). The process is running in the background.
T Stopped, either by a job control signal or because it is being traced. The process is suspended.
W paging (not valid since the 2.6.xx kernel)
X dead (should never be seen)
Z Defunct ("zombie") process, terminated but not reaped by its parent.
